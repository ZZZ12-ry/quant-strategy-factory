## 功能模块

### 1. 主页面 (app.py)

**策略回测**
- 参数配置区（策略类型、周期、品种、手续费）
- 一键运行回测
- 关键指标卡片（总收益/年化/夏普/回撤/胜率）
- 累计收益对比曲线
- 回撤曲线
- 月度收益热力图
- 交易记录表格

**侧边栏导航**
- 页面切换（策略回测/特征工程/模型训练/IC 分析/关于）
- 快速操作按钮
- 系统状态显示

### 2. 特征工程页面 (02_feature_engineering.py)

**功能**
- 特征选择方法选择（7 种）
- 阈值/参数配置
- 特征数量统计
- 相关性热力图（Plotly 交互式）
- 特征重要性排名
- 特征统计信息表

**集成**
- 调用 `src.ml.auto_feature_selector`
- 实时计算并展示结果

### 3. IC 分析页面 (03_ic_analysis.py)

**功能**
- 关键指标卡片（IC/Rank IC/ICIR/方向准确率）
- IC 时间序列图（双指标对比）
- IC 分布直方图
- 分层回测柱状图（5 组 + 多空收益）
- 分层回测数据表
- 评估结论（优秀/可用/需改进）
- IC 指标说明文档

**集成**
- 调用 `src.ml.model_ic_evaluator`
- 自动计算评估结论

## 使用指南

### 启动方式

**方法 1: 一键启动**
```bash
run_ui.bat
```

**方法 2: 手动启动**
```bash
pip install streamlit plotly
cd ui
streamlit run app.py
```

### 访问地址

- 本地：http://localhost:8501
- 网络：http://192.168.1.115:8501

## 核心优势

### 对比命令行

| 维度 | 命令行 | UI 界面 | 提升 |
|------|--------|--------|------|
| **上手难度** | 需要 Python 基础 | 点击按钮即可 | 大幅降低 |
| **结果展示** | 文本/JSON | 交互式图表 | 直观清晰 |
| **参数调整** | 改代码重跑 | 滑块实时调 | 效率提升 10 倍 |
| **可视化** | 需额外代码 | 内置图表 | 开箱即用 |
| **分享演示** | 难 | URL 分享即可 | 便于协作 |

### 对比完整前端

| 维度 | Vue/React | Streamlit | 选择理由 |
|------|-----------|-----------|---------|
| **开发时间** | 2-3 周 | 1 天 | 快 15 倍 |
| **技术栈** | 前后端分离 | 纯 Python | 无需前端 |
| **部署** | 复杂 | 一条命令 | 简单 |
| **迭代** | 需编译构建 | 修改即生效 | 快速 |
| **维护** | 中 | 低 | 省力 |
